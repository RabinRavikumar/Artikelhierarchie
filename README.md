# Artikelhierarchie
Dieses Projekt ist ein Java-OOP-Schulprojekt zur Verwaltung von Artikeln.
Es enthält eine Oberklasse Artikel mit Unterklassen für Elektronik (Digitalkamera, DVDPlayer, Fernseher) und Möbel (Tisch, Schrank, Sofa).
Zusätzlich gibt es Klassen wie Angebot und Inventur, um mehrere Artikel zusammenzufassen und zu verwalten.
Ziel des Projekts ist es, die Prinzipien der objektorientierten Programmierung wie Vererbung praktisch zu üben.
